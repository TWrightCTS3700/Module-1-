## Project Description

This Java project demonstrates object-oriented programming by using two sport-related classes: Basketball and Football. The program creates objects from each class, displays their original values, updates their values using methods, and then prints the updated information.

## What the Project Does

The program starts in the SportsApp class. It creates one Basketball object and one Football object. The Basketball object stores a team name, number of three-pointers made, and number of fouls. The Football object stores a team name, number of touchdowns, and passing yards.

The program displays the original values, calls methods that update the objects, and then displays the updated values.

## Important Programming Concepts Used

- Classes and objects
- Constructors
- Overloaded constructors
- Private fields
- Getter and setter methods
- Custom methods
- Object state updates
- Console output

## Expected Inputs

This program does not require user input. The values are written directly in the code.

## Expected Outputs

The program prints basketball and football information before and after updates. It shows the team names, three-pointers made, fouls, touchdowns, and passing yards.
